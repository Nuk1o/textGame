# TextGame
## RUS
  Создать консольную игру.
  
  Команды игрока:
- осмотреться
- идти <имя локации>
- взять <имя предмета>

  Локации:
- кухня
- коридор
- комната
- улица

  Предметы:
- рюкзак
- конспекты
- ключи
